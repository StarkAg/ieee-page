# One Piece Startup Summit 2026

Landing page for the IEEE CS SRM One Piece themed startup pitch event, "Treasure Hunt for Unicorns".

Live site: https://one-piece-startup-summit-2026.vercel.app

## Stack

- React
- Vite
- GSAP
- Responsive video backgrounds

## Highlights

- One Piece inspired event landing page styling
- Persistent desktop and mobile background video support
- Responsive fixed header with mobile navigation
- Rules, schedule, mechanics, awards, and registration sections
- Tuned animations for lower runtime lag

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```
